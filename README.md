# Muhammad Abu-Bakr - Professional Portfolio

A modern, multi-page portfolio website built with React, Tailwind CSS, and Supabase.

## Features

- 🎨 Modern, editorial design with dark/light mode
- 📱 Fully responsive across all devices
- ✨ Smooth animations with Framer Motion
- 📝 Blog system with markdown support
- 💼 Project portfolio with case studies
- 📊 View counters for blog posts and projects
- 🔍 Search and category filtering
- 📧 Contact form with Supabase storage
- 🔐 Admin panel for content management
- 🎯 SEO optimized
- 🖱️ Custom animated cursor

## Tech Stack

- **Frontend**: React 18, Tailwind CSS, Framer Motion
- **Backend**: Supabase (PostgreSQL, Authentication, Storage)
- **Deployment**: Vercel/Netlify
- **Images**: Stored in GitHub (/public folder)
